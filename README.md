# movie_ticket
Project Title

Movie Ticket Booking System (Python)

Features
User se ticket type input leta hai.
Silver, Gold aur Platinum tickets support karta hai.
Invalid ticket type ko handle karta hai.
Multiple tickets book karne ki facility deta hai.
Total bill automatically calculate karta hai.
Uses Python Dictionary, Loops, Conditions, and User Input.

Python Dictionaries
while Loop
if-else Conditions
User Input (input())
String Methods (lower(), strip())
Arithmetic Operations
Loop Control (break)
